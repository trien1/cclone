# installation instructions

1. clone the repository
```
git clone https://github.com/trien1/cclone.git
```
cd into the directory
```
cd cclone/
```
3. give the needed permission to execute the script
```
chmod +x install.sh
```
4. run the script
```
./install.sh
```
